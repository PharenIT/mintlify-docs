Die Pharen REST API gibt vertrauenswürdigen Backend-Services, Sync-Jobs und internen Tools Zugriff auf Pharen Hub. Die API-Referenz nutzt den interaktiven Mintlify-Playground, damit du Parameter prüfen, Beispiele generieren und API-Key-Anfragen direkt ausprobieren kannst.

## Basis-URL

```bash
https://pharen.app/api
```

Im API-Playground ist `https://pharen.app` standardmassig ausgewahlt. Nutze den Base-URL-Selector, wenn du gegen eine andere Umgebung testen willst, zum Beispiel gegen ein lokales Backend unter `http://localhost:8000`.

## Authentifizierung

Erstelle unter **Einstellungen -> API-Keys** einen begrenzten Key und sende ihn als Bearer-Token:

```bash
Authorization: Bearer phk_...
```

Wenn dein Client keinen `Authorization`-Header setzen kann, nutze stattdessen `X-API-Key`.
